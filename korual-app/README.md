# KORUAL 통합 운영 프로그램

관제 대시보드, 워크플로 보드, AI 브리핑, 데이터, 분석, 알림, 관리자 기능을 하나로 통합한 React 웹 프로그램입니다.

## 실행

```powershell
cd korual-app
npm install
npm run dev
```

## 프로덕션 빌드

```powershell
npm run build
npm run preview
```

## 구현 기능

- 실시간 운영 KPI, 추이 차트, 활동, 우선순위, 시스템 상태
- 상태별 워크플로 보드, 새 업무 추가, 상세 패널, 단계 이동
- AI 우선순위 브리핑, 이상 징후, 권장 조치 실행, 운영 어시스턴트
- 전체 검색과 워크플로 데이터 테이블
- 상태별 업무 및 팀 성과 분석
- 알림 읽음 처리와 미확인 개수 동기화
- 사용자 권한과 시스템 설정 화면
- 브라우저 localStorage 기반 업무·알림 상태 유지
- 데스크톱, 태블릿, 모바일 반응형 레이아웃

## 검증

로컬 작업 환경에서 `npm run build`와 정적 서버 HTTP 200 응답을 확인했습니다.
